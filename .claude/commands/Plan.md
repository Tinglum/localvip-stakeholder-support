# /plan

Create a concise implementation plan before coding.

## Steps

1. Inspect relevant files
2. Identify the real goal
3. List constraints
4. Define the smallest useful implementation slice
5. Identify files likely to change
6. State verification steps

## Output

Return:
- Goal
- Relevant files
- Proposed approach
- Risks
- First implementation step

Keep it concise.
Do not write code unless asked.
